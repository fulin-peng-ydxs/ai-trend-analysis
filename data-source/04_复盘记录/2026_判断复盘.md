# 2026判断复盘

> 建立日期：2026-06-20
> 定位：年度复盘，不构成投资建议。

## 1. 今年的核心假设

2026 年 AI 投资主线不是从“有没有 AI 需求”开始，而是从“AI 需求继续增长后，哪一层基础设施最先被卡住”开始。

当前核心假设：

AI 使用量继续增长
-> 推理需求继续增长
-> 数据中心继续扩张
-> 网络、电力、散热和交付周期成为更重要瓶颈
-> 资本开支继续向基础设施外溢

## 2. 当前判断记录

| 方向 | 当前判断 | 置信度 | 关键验证信号 |
| --- | --- | --- | --- |
| 推理基础设施 | 2026 年主线之一，需求确定性较高 | 高 | 云厂商资本开支、AI 收入、推理芯片出货 |
| 电力与散热 | AI 数据中心扩张后的硬约束 | 高 | 并网周期、变压器订单、液冷渗透率 |
| CPO与光互连 | GPU 集群扩大后的二级瓶颈 | 中高 | 1.6T 放量、CPO 量产、毛利率 |
| Agent | 推理需求放大器，但应用层需验证收入 | 中 | 企业付费、使用量、算力消耗 |
| 机器人 | 长期方向，2026 年偏观察 | 中低 | 批量订单、场景 ROI、零部件瓶颈 |

## 3. 需要避免的错误

- 把“AI 相关”当成“业绩受益”；
- 只看趋势，不看瓶颈；
- 只看瓶颈，不看谁付钱；
- 只看订单，不看利润和供给；
- 只看短期涨幅，不看是否已经充分定价；
- 把 demo 当成商业化；
- 把一次性项目收入当成长期成长。

## 4. 复盘模板

每个季度复盘一次，用下面格式：

```md
## 被证伪的部分

```

## 5. 2026Q3待复盘问题

### 推理基础设施

- 云厂商是否继续上修资本开支？
- 推理收入是否开始更清晰地披露？
- GPU 与自研芯片是否共同增长，还是出现替代关系？

### 电力与散热

- 数据中心建设是否继续受制于电力？
- 液冷是否从部分高密度机柜扩展到更广泛部署？
- 电力设备公司订单是否能转化为利润？

### CPO与光互连

- 1.6T 是否出现更明确订单？
- CPO 是否有实质性采购，而不是技术路线讨论？
- 光模块公司毛利率是否保持？

### Agent

- Agent 是否带来可量化付费？
- Agent 是否真正增加推理 token 消耗？
- 企业是否愿意把核心流程交给 Agent？

### 机器人

- 是否出现批量采购？
- 真实部署成本是否下降？
- 供应链中哪一环最先紧张？

## 6. 年底复盘要回答的问题

到 2026 年底，至少要回答：

- 今年最强的 AI 产业瓶颈是什么？
- 哪些瓶颈真的变成了资本开支？
- 哪些资本开支传导到了上市公司收入和利润？
- 哪些方向只是叙事，没有验证？
- 哪个判断最成功？
- 哪个判断偏差最大？
- 偏差来自信息不足、框架错误，还是市场定价错误？
