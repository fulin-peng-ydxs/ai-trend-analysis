# 2026Q3观察清单

> 建立日期：2026-06-20
> 定位：季度观察，不构成投资建议。

## 1. 本季度核心问题

2026Q3 最重要的问题不是“AI 还热不热”，而是：

AI 基础设施资本开支是否继续上修，以及新增资本开支流向哪一层瓶颈。

## 2. 观察方向一：推理基础设施

### 要验证的问题

- 推理需求是否继续高增长；
- AI 收入是否能覆盖更高折旧、电力和芯片成本；
- 单 token 成本下降是否带来更大使用量；
- 自研芯片是否开始分流 GPU 需求。

### 跟踪信号

- NVIDIA 数据中心收入和管理层对推理的表述；
- Microsoft AI 年化收入运行率、Azure 增速、AI 基础设施投入；
- Amazon AWS 增速、Trainium 和 NVIDIA GPU 部署；
- Alphabet Google Cloud 与 AI Infrastructure 表述。

## 3. 观察方向二：Agent

### 要验证的问题

- Agent 是 demo 热，还是已经带来真实付费；
- Agent 是否显著放大推理调用；
- 企业是否愿意为 Agent 单独付费；
- 权限、安全、审计是否成为采购刚需。

### 跟踪信号

- 企业 Agent 产品的付费用户和留存；
- 编程 Agent、办公 Agent 的使用时长；
- 云厂商是否披露 Agent 带来的算力需求；
- 大模型公司是否把 Agent 作为核心收入驱动。

## 4. 观察方向三：CPO与下一代光互连

### 要验证的问题

- 800G 需求是否延续；
- 1.6T 是否开始规模放量；
- CPO 和硅光是否从验证进入采购；
- 毛利率是否被竞争压缩。

### 跟踪信号

- 光模块公司订单和出货；
- 上游激光器、DSP、硅光芯片供需；
- 云厂商数据中心网络架构升级；
- NVIDIA、交换芯片厂商对下一代互连的路线表述。

## 5. 观察方向四：电力与散热

### 要验证的问题

- 电力是否继续成为数据中心建设瓶颈；
- 液冷是否从高端方案变成主流配置；
- 变压器、UPS、配电设备是否继续供给紧张；
- 数据中心项目是否因为并网和审批延迟。

### 跟踪信号

- 云厂商资本开支指引；
- 数据中心项目开工、延期和并网消息；
- 电力设备交期；
- 液冷设备订单；
- 电价和监管政策变化。

## 6. 观察方向五：机器人

### 要验证的问题

- 是否出现批量订单；
- 场景 ROI 是否清晰；
- 整机成本是否下降；
- 零部件是否成为瓶颈。

### 跟踪信号

- 工厂、仓储、物流机器人部署；
- 关键零部件订单；
- 整机厂交付数量；
- 大客户采购和合作。

## 8. 当前优先级

高优先级：

- 推理基础设施；
- 电力与散热；
- CPO 与下一代光互连。

中优先级：

- Agent；
- 机器人。

低优先级：

- 纯概念应用；
- 没有订单验证的主题炒作；
- 只讲空间、不讲资本开支传导的方向。
